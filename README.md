# AirHub-Beijing · 京翼机场中枢

> 一个从 0 到 1 动手制作的数据可视化大屏开源项目。

京翼机场中枢（AirHub-Beijing）是一个公开的数据大屏示例项目，旨在帮助开发者通过实践理解数据可视化大屏的完整构建流程：从数据获取、处理、布局设计到组件渲染与交互。

## 项目简介

- **项目英文名**：AirHub-Beijing
- **项目中文名**：京翼机场中枢
- **类型**：数据可视化大屏（Data Visualization Dashboard）
- **目标**：以机场运行场景为背景，逐步构建一套可用于学习的大屏模板
- **开源协议**：[MIT](./LICENSE)

## 功能规划

> 以下功能将随项目迭代逐步完善。

- [ ] 大屏整体布局与自适应缩放
- [ ] 航班起降 / 吞吐量实时数据展示
- [ ] 客流、行李、安检等运行指标看板
- [ ] 地图与航线可视化
- [ ] 图表组件（折线、柱状、饼图、雷达等）
- [ ] 数据模拟与接口对接示例
